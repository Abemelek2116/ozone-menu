# Ozone Menu

Ozone Menu is a modern, responsive digital restaurant menu platform designed for restaurants, cafés, hotels, bars, and lounges.

Customers can instantly access the menu by scanning a QR code, while restaurant owners manage menu items, categories, pricing, availability, and settings through a secure web-based dashboard powered by Google Sheets and Google Apps Script.

## Features

- 🍽️ QR Code Digital Menu
- 📱 Fully Responsive Design
- ⚡ Fast Static Website
- 🔍 Search & Category Filtering
- 🏷️ Featured & Popular Items
- 💰 Real-Time Price Updates
- 🖼️ High-Quality Food Galleries
- 🌐 Multi-language Ready
- 📊 Owner Analytics Dashboard
- 🔐 Google Sign-In Authentication
- ☁️ Google Sheets Content Management
- 🚀 Easy Deployment with GitHub Pages

## Technology

- HTML5
- CSS3
- JavaScript (ES6)
- Google Apps Script
- Google Sheets (Database/CMS)
- GitHub Pages
